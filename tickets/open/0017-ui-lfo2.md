## Summary

Surface the E004 additions on the editor's per-layer faceplate: a second LFO
section (shape / rate / delay), the new sixth matrix row (Lfo2 → pitch / cutoff
/ amp / PWM), and the per-LFO sync + reset toggles for both LFOs. Rides the
existing Upper/Lower layer-toggle UI from E003 (ticket 0013).

## Acceptance criteria

- [ ] LFO2 controls (shape selector, rate, delay) mirror the LFO1 section, bound
      to the `Lfo2*` params, on the per-layer faceplate.
- [ ] Modulation matrix view shows 6 source rows × 4 destinations; the Lfo2 row
      edits the four new depth params.
- [ ] Per-LFO sync toggle (`LfoSync` / `Lfo2Sync`) and reset toggle (`LfoReset`
      / `Lfo2Reset`) for each LFO.
- [ ] When an LFO's sync is on, its rate control displays musical subdivisions
      (note values) instead of Hz; off shows Hz. (Depends on 0015's param shape.)
- [ ] Controls respect the Upper/Lower edit target and are hidden/handled in
      Whole mode consistently with the rest of the faceplate.
