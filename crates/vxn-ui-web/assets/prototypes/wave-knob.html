VXN1 — Wave Knob Prototype

Single SVG per knob: knob face + rotating indicator + 4 glyph labels at the four corner positions (SW, NW, NE, SE) outside the rim. Drag up = CW, drag down = CCW (vertical motion only — 30 px per detent). Indicator is clamped at SW (start) and SE (end), no wrap-around. Click a glyph to jump directly. Indicator picks the shortest rotation path so consecutive moves look like a real knob, not a CSS rotate(0deg → 270deg) regression.
